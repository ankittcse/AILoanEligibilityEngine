
This project uses Vite, React, and Supabase for authentication.

## Local setup

1. Copy `.env.example` to `.env`
2. Fill in your Supabase details:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
3. Run:
   ```bash
   npm install
   npm run dev
   ```

## Supabase env example

See `.env.example` for the required environment variables.

## OpenAI setup

If you want the chat assistant to answer questions with real AI instead of demo responses, add:

```env
VITE_OPENAI_API_KEY=your-openai-api-key
VITE_OPENAI_MODEL=gpt-3.5-turbo
```

## Gemini setup

If you want to use Google Gemini instead of OpenAI, add:

```env
VITE_GEMINI_API_KEY=your-gemini-api-key
VITE_GEMINI_MODEL=gemini-1.5-turbo
```

Gemini is automatically used when `VITE_GEMINI_API_KEY` is present.

## Free/demo mode

If neither `VITE_OPENAI_API_KEY` nor `VITE_GEMINI_API_KEY` is configured, the chat assistant will fall back to a free demo response mode. This means the app still works, but responses are simulated rather than generated by a live AI model.

> Important: start the app with `npm run dev` for the Gemini proxy to work locally. If you use `npm run preview`, the Vite preview server does not include the same dev proxy middleware.

Then restart the dev server.
